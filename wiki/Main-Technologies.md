# Main Technologies

The WebApp builds on a curated set of modern web technologies:

- **Vue 3**: The core reactive UI framework (Composition API).
- **Vite 5**: The bundler and hot-reloading development server.
- **PrimeVue 4**: The UI component library providing consistent grids, modals, and input controls.
- **Pinia 2**: Lightweight, reactive state management.
- **Axios 1**: Promise-based HTTP client to make backend connections.
- **Vue Router 4**: Declarative client routing.
- **Vue I18n 9**: Localization support (English and Spanish translations).
