# Project Overview

Nexa is a B2B cold-chain operations platform for importer-distributors, commercial teams, logistics operators, warehouse users, and B2B buyers.

The WebApp provides:

- authenticated role-based entry,
- commercial dashboards,
- catalog and product visibility,
- B2B client management,
- purchase requests and sales orders,
- inventory and stock movements,
- dispatch and delivery tracking,
- analytics and reports,
- buyer portal workflows.

The WebApp is separate from `nexa-platform`. It consumes the backend through REST endpoints under `/api/v1`.

## Repositories

| Repository | Role |
| --- | --- |
| `nexa-website` | Static landing website |
| `nexa-webapp` | Angular WebApp and buyer portal |
| `nexa-platform` | Java Spring Boot REST API |
