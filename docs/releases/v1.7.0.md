# Nexa WebApp v1.7.0 - Hybrid API Integration & Responsive Polish

## Overview
This release closes the technical integration for AV2. It introduces a hybrid data strategy combining the 73 real .NET backend endpoints with kebab-case mock routing, provides a complete mobile/desktop layout review, and completes terminology alignment.

## Main Changes

### Hybrid API Integration
- **Dual Source Configuration:** Separated API endpoints by binding core modules to `VITE_NEXA_API_BASE_URL` (C# Backend) and remaining modules to `VITE_NEXA_MOCK_API_BASE_URL` (JSON Server).
- **Mock Routing:** Fixed kebab-case routes for purchase requests, alerts, temperature logs, and timeline events.

### UI Polish
- **Responsive Layouts:** Resolved overflow and alignment issues on Inventory Control, Commercial Dashboard, and Product Catalog.
- **Localization:** Consolidated English and Spanish translations for all core domain terms.
