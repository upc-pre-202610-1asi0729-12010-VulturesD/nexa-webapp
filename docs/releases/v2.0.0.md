# Nexa WebApp v2.0.0

Open Source delivery release for the Nexa Angular WebApp.

## Scope

- Angular 18 single-page application.
- TypeScript and Angular Material UI.
- Bounded-context structure under `src/app`.
- REST API integration with Nexa Platform under `/api/v1`.
- Production API base URL configured in `src/environments/environment.prod.ts`.
- Render Static Site deployment with SPA fallback.
- Buyer portal, commercial, logistics, warehouse, analytics, and document workflows.

## Deployment

Render Static Site configuration is defined in `render.yaml`:

- build command: `npm ci && npm run build`
- publish path: `dist/nexa-webapp/browser`
- SPA fallback: `/*` to `/index.html`
- Node version: `20`

## Validation

```bash
npm ci
npm run build
```
