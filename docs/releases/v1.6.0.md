# Nexa WebApp v1.6.0 - UI Layout Refactoring & Data Store Centralization

## Overview
This release focuses on refactoring the Vue layout structure, modernizing typography, and centralizing data store operations to support the hybrid data delivery model.

## Main Changes

### Layouts & Styling
- **Modern Layouts Grid:** Refactored ops and portal layouts to leverage flexible CSS layouts.
- **Typography & Polish:** Integrated Outfit/Inter fonts and normalized scroll behavior.

### Data Store
- **Centralized Data Store:** Refactored `data.store.js` to manage initial loading states, separating local mock collections from direct backend endpoint bindings.
