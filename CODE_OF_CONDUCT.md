# Code of Conduct

This repository is maintained for an academic software engineering project.

Contributors should:

- Keep discussion respectful and technical.
- Review changes by scope and evidence, not by personal preference.
- Avoid publishing secrets, private credentials, or unrelated files.
- Keep feedback actionable and tied to the WebApp requirements.

Maintainers may close issues or pull requests that are abusive, off-topic, or unsafe for the project.
